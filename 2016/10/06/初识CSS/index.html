初识CSS

块级元素：
<div>、 <p>、<h1>、<form>、<ul>和<li>…
a {display:block;}可将内联元素转换成块状元素

块状元素的特点：
1、每个块级元素都从新的一行开始，并且其后的元素也另起一行。（真霸道，一个块级元素独占一行）
2、元素的高度、宽度、行高以及顶和底边距都可设置。
3、元素宽度在不设置的情况下，是它本身父容器的100%（和父元素的宽度一致），除非设定一个宽度。

内联元素：
<span>、<a>、<label>、 <strong>…
div {display:inline;} 可将块状元素转换成内联元素

内联元素特点：不可换行，不可设置宽、高度

内联块状元素：
<img>、<input>…

a {display:inline-block;}可将内联元素转换成内联块状元素

内联块状元素特点：
1、不换行
2、高、宽度可设置

盒子模型：
块状元素的p、h、table、div…都具备盒子模型

盒子实际内容：元素内容content、内边距padding、元素边框border、外边距margin

一个盒子实际宽度=content的宽度+padding的宽度2+border的宽度2+margin的宽度*2

盒模型—border：
可设置它的粗细、样式、颜色三个属性
exp：
div{ border-width:2px; border-style:solid; border-color:red; }
也可简写：
div{ border:2px solid red; }

网页布局——三种布局模型：
1、流动模型(Flow)

流动模型是网页默认的布局方式，块级元素自上而下独占一行，宽度默认是100%，内联元素自左向右

2、浮动模型(Float)

要使块级元素同行显示，设置float和适应的宽度即可

3、层模型(Layer)

a、绝对定位(position: absolute)

使元素脱离文档流，若无已定位的父级元素，则相对于<html>标签进行定位；若有已定位的父级元素，则相对于最接近的父元素进行绝对定位

b、相对定位(position: relative)
元素不脱离文档流，定位后保留原来的位置

c、固定定位(position: fixed)
不会随浏览器窗口的滚动而变化

内联元素居中：
在父元素的样式添加 text-align：center；

定宽块级元素居中：
margin：0 auto；

不定宽块级元素居中：

a、lable标签有长度自适应性，可看成是默认定宽元素
display:lable; margin:0 auto;
d、换成内联元素，但高宽度不可设置
display:inline; text-align:center;
c、
父元素{float:left;position:relative;left:50%;}
子元素{position:relative;left:-50%;}

垂直居中：

父元素高度确定的单行文本：
设置父元素的 height 和 line-height 高度一致，line-height 与 font-size 的计算值之差，在 CSS 中成为“行间距”。分为两半，分别加到一个文本行内容的顶部和底部。

父元素高度确定的多行文本：

a、css 中有一个用于竖直居中的属性 vertical-align，在父元素设置此样式时，会对inline-block类型的子元素都有用。
table 、tbody、tr、中的td标签默认设置了vertical-align：middle；只要添加相应标签即可

b、父元素{ height:300px; background:#ccc; display:table-cell; /*IE8以上及Chrome、Firefox*/ vertical-align:middle; /*IE8以上及Chrome、Firefox*/ }

隐性改变display类型：

  1. position : absolute

  2. float : left 或 float:right

简单来说，只要html代码中出现以上两句之一，元素的display显示类型就会自动变为以 display:inline-block（块状元素）的方式显示，当然就可以设置元素的 width 和 height 了，且默认宽度不占满父元素。

CSS 规定的定位机制有三种，分别是标准文档流、浮动及绝对定位。

CSS中position属性有四种：static（正常文档流）、relative（相对定位）、absoulute(绝对定位)、fixed（固定定位）。
