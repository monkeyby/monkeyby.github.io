jQuery摘记（三：事件篇）

click事件：

1、$ele.click( handler(eventObject) )

2、$ele.click( [eventData ], handler(eventObject) )

mouse事件：
1、$ele.mousedown( handler(eventObject) )

2、$ele.mousedown( [eventData ], handler(eventObject) )

提示：mouseup、mousemove、mouseover、mouseout、keydown、keyup、keypress同理，不再赘述,mouseenter、mouseleave不会冒泡处理问题

.hover(function1,function2) //使用hover方法，鼠标移入时触发func1，移除触发func2

.focusin() //鼠标聚焦，多用于input

.focusout() //鼠标点击失去焦点

注意：focusin和focuout支持冒泡，blur和focus不支持冒泡

.change() //监听input、textarea、select是否改变

.select() //当元素的value被选中时触发

.submit() //提交表单时触发

on的多事件绑定：

基本用法：.on( events ,[ selector ] ,[ data ] )

$(“#elem”).click(function(){}) //快捷方式
$(“#elem”).on(‘click’,function(){}) //on方式

多个事件绑定同一个函数
$(“#elem”).on(“mouseover mouseout”,function(){ });

多个事件绑定不同函数
$(“#elem”).on({
mouseover:function(){},
mouseout:function(){},
});

on()的高级用法：on的委托机制
.on( events ,[ selector ] ,[ data ], handler(eventObject) )

$(“div”).on(“click”,”p”,fn) //事件绑定在最上层div元素上，当用户触发在a元素上，事件将往上冒泡，一直会冒泡在div元素上。如果提供了第二参数，那么事件在往上冒泡的过程中遇到了选择器匹配的元素，将会触发事件回调函数

.off() //不加参数默认删除所有事件

event.type：获取事件的类型

event.pageX 和 event.pageY：获取鼠标当前相对于页面的坐标

event.preventDefault() 方法：阻止默认行为

event.stopPropagation() 方法：阻止事件冒泡

event.which：获取在鼠标单击时，单击的是鼠标的哪个键

event.currentTarget : 在事件冒泡过程中的当前DOM元素

this和event.target的区别：
js中事件是会冒泡的，所以this是可以变化的，但event.target不会变化，它永远是直接接受事件的目标DOM元素；
