jQuery摘记（四：动画篇）

$elem.hide(time,callback) //不加参数直接隐藏，可加fast/slow/具体时间，也可加回调函数
$elem.show(time,callback) //与hide同理

$elem.toggle() //在hide和show之间切换

.slideDown(time,callback) //用滑动动画显示一个匹配元素
.slideUp(time,callback) //与slideDown同理
.slideToggle(time,callback) //在slideDown与slideUp之间切换

fadeOut(time,callback) //淡出
fadeIn(time,callback) //淡入
fadeToggle(time,callback) //淡入与淡出切换
fadeTo(time,opacity,callback) //自定义淡出淡入的透明度

1、stop() 动画队列如果有多个动画,停止当前执行的动画,开始下一个动画

2、stop(true) 真正的停止,当前动画停止,队列里后续动画也不执行

3、stop(true,true) 当前动画动作停止，并且直接展示所有动画完成后的最终状态

jQuery.each(array, callback )
jQuery.each(object, callback )

jQuery.inArray( value, array ,[ fromIndex ] ) //查找value在array中的位置，可以通过fromIndex规定查找的起始值，默认数组是0开始

jQuery.trim() //用于去除字符串两端的空白字符

.get( [index ] ) //单独获取合集中的的某一个元素

.index() //语法：参数接受一个jQuery或者dom对象作为查找的条件，返回集合的索引值

.animate( properties ,[ duration ], [ easing ], [ complete ] )
.animate( properties, options )

properties 对象{},注意：所有指定的属性必须用骆驼形式
duration (可选)三种预定速度之一的字符串(“slow”, “normal”, or “fast”)或表示动画时长的毫秒数值(如：1000)
easing (可选)String要使用的擦除效果的名称(需要插件支持).默认jQuery提供”linear” 和 “swing”
complete (可选)Function在动画完成时执行的函数
