DOM中clientTop、scrollTop、offsetTop的知识

1、元素的偏移量

offsetHeight: 元素在垂直方向上占用空间的大小，以像素计算。
offsetWidth: 元素在水平方向上占用的空间的大小，以像素计算。
offsetLeft： 元素的左外边框至包含元素的左内边框之间的距离。
offsetRight: 元素的右外边框至包含元素的右内边框之间的距离。

2、元素的客户端大小

clientHeight = content + padding
clientWidth = content + padding
clientTop = 元素的上边框厚度
clientLeft = 元素的左边框厚度

3、元素滚动大小

scrollHeight: 在没有滚动情况下，元素内容的总高度
scrollwidth: 在没有滚动情况下，元素内容的总宽度
scrollLeft: 被隐藏的左侧内容的像素数，通过设置这个值可以改变元素的滚动位置
scrollTop: 被隐藏的上侧内容的像素数，通过设置这个值可以改变元素的滚动位置

4、鼠标坐标

event.clientX: 相对文档的水平坐标
event.clientY: 相对文档的垂直坐标
event.offsetX: 相对容器的水平坐标
event.offsetY: 相对容器的垂直坐标

5、IE、FF的兼容性：

IE6.0、FF1.06+

clientWidth = width + padding
clientHeight = height + padding
offsetWidth = width + padding + border
offsetHeight = height + padding + border

IE5.0/5.5

clientWidth = width - border
clientHeight = height - border
offsetWidth = width
offsetHeight = height

(注意：IE6一下width和height包括padding和border；margin值与上述的全无关系)
